Protección Inteligente con Tecnología Láser

Sistema de detección de intrusos rápido, preciso y accesible para todos.

Introducción

Laser Lock es un sistema de seguridad láser diseñado para hogares y pequeños negocios. Utiliza un haz de luz y espejos estratégicos para detectar cualquier intrusión de forma inmediata y confiable. Su diseño se centra en la accesibilidad, bajo costo y facilidad de instalación.

Problema y Objetivos de Desarrollo Sostenible

El Problema

Los sistemas de seguridad convencionales suelen ser costosos y requieren instalación compleja. Esto deja a familias y pequeños negocios en vulnerabilidad frente a robos.

ODS Relacionados
  • ODS 9: Promover tecnologías accesibles y sostenibles.
  • ODS 11: Garantizar comunidades seguras y resilientes.

Nuestra Solución

Laser Lock integra un sistema físico basado en láseres y espejos de 2x2 cm con una plataforma web para monitoreo remoto. Gracias a la conectividad WiFi del módulo ESP8266, se pueden recibir alertas en tiempo real en cualquier dispositivo.

Tecnologías Involucradas

Software
  • CSS, HTML, JavaScript, C++
Hardware
  • ESP8266 NodeMCU
  • Láser compatible con ESP8266
  • Sensor de luz, Teclado 4x4, Pantalla LCD
  • Chapa eléctrica, Buzzer, LED bicolor
  • 10 espejos de 2x2 cm
